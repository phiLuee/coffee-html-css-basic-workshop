# Coffee Workshop: HTML & CSS Basics

Welcome to the repository for our HTML & CSS Basics Workshop! This project is designed to introduce you to the fundamental concepts of web development.

## 🚀 About this Workshop

The goal of this workshop is to provide a hands-on introduction to HTML (HyperText Markup Language) and CSS (Cascading Style Sheets). By the end of this session, you will:

- Understand the basic structure of an HTML document.
- Know how to use common HTML tags to create content.
- Learn how to apply CSS to style your webpage, including colors, fonts, and layout.
- Have built a simple, yet beautiful, webpage from scratch.

## ☕ The Project

In this workshop, we will be building a simple, single-page layout for a fictional coffee shop. This repository contains the starting files and assets (like images) you'll need to follow along.

## 🛠️ Getting Started

To get started, simply clone this repository to your local machine:

```bash
git clone https://github.com/phiLuee/coffee-html-css-basic-workshop.git
```

Then, open the `index.html` file in your browser and your favorite code editor to start building!
